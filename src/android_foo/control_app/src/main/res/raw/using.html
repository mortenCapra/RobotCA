Using

Once connected, a variety of screens and control modes can be chosen from to operate the robot.

Controlling the Robot

Several control schemes exist for moving the robot and can be selected in the drop down menu in the control robot screen.

Joystick Control

In this mode the robot can be controlled with the joystick on the bottom right of the screen.

Tilt Control

In this mode, on devices with accelerometers, the robot can be controlled by tilting the android device. Tilt forward or backwards to drive forward or reverse and left or right to turn. Tapping the joystick will recalibrate the tilt sensor.

Routing

In this mode, the robot will follow the trail marked by the placed Waypoints. See the Map view for placing and viewing Waypoints.

Random Walk

In this mode, the robot drives at random in an area. See the Map view for placing and viewing areas. if the robot encounters an obstacle, it will steer around it.

Obstacle

This mode is not for controlling the robot directly, but for creating obstacles the robot will avoid in the different controlmodes. See the Map view for placing and viewing obstacles.

Navigation Drawer Views

Once connected to the robot, the navigation drawer on the left can be opened to choose from the following screens:

The Control Robot

The control robot screen, is the main screen in which the robot is controlled. This screen contains the Map screen and the Camera screen

The Camera screen

The Camera screen shows the view from the robot's camera.

The Map screen

The Map screen shows both the robot's and android device's GPS position on the map. This is the screen used to control the robot using GPS location and the different controlmodes. Points are set via long press, and will have different meanings depending on the controlmode. The points can be moved via long press on existing points

When Routing is selected, the points will form a route which the robot will follow. The robot will steer around obstacles placed

When Random walk is selected the points will form an area, in which the robot will randomly move

When Obstacle is selected the points will from obstacles, which the robot will steer around in either Routing or Random walk

Preferences

The Preferences allows you to edit a variety of settings including the ROS topics you are connected to and warning options.
